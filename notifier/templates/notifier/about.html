About

Having tried to buy products from the Dell Outlet with some dificulty in the past, I wrote a small script that would check the stock status of the laptop I wanted, allowing me to avoid manually refreshing the page to check stock levels throughout the day

This script worked well, and there didn't seem to be any other tools online that offered this service - so OutletMeKnow was born!

The service is written in Python, using Django, Celery and Postgresql as a database server.

I wrote most of this thing on a saturday night after drinking a glass or two of wine, so the code leaves a little to be desired. It's all open source, so if you'd like to contribute then check it out on GitHub.
